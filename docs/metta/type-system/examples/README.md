# MeTTa Type System Examples

This directory contains executable examples demonstrating MeTTa's type system features.

## Examples

1. **01-basic-types.metta** - Basic type annotations and function types
2. **02-type-checking.metta** - Pragma-controlled type checking
3. **03-gradual-typing.metta** - %Undefined% and gradual type introduction
4. **04-polymorphism.metta** - Type variables and generic functions
5. **05-dependent-types.metta** - Length-indexed vectors
6. **06-type-operations.metta** - Runtime type introspection
7. **07-type-errors.metta** - Type error messages
8. **08-meta-types.metta** - Meta-types and evaluation control

## Running Examples

```bash
# Using metta executable
metta 01-basic-types.metta

# Using Python
python3 -m hyperon.runner 01-basic-types.metta
```

## Prerequisites

- hyperon-experimental installed
- Python 3.8+ (for Python runner)

## Learning Path

1. Start with **01-basic-types.metta** for fundamentals
2. Learn type checking with **02-type-checking.metta**
3. Understand gradual typing: **03-gradual-typing.metta**
4. Explore polymorphism: **04-polymorphism.metta**
5. Advanced: **05-dependent-types.metta**, **08-meta-types.metta**
